# 🚀 Developer Portfolio

A modern, responsive full-stack developer portfolio built with React and Vite, designed to showcase my projects, skills, and experience through a clean UI and smooth animations.

---

## 🌐 Live Demo

👉 https://my-portfolio-website-kappa-orcin.vercel.app/

---

## 📌 Features

- ⚡ Modern and responsive design
- 🎨 Clean UI with dark theme and gradient styling
- 🧠 Skills section with categorized technologies
- 💼 Projects showcase with live demos and GitHub links
- 📩 Contact section with functional form
- ✨ Smooth animations using Framer Motion
- 📱 Fully mobile-friendly

## ⚙️ Installation & Setup

- Clone the repository:

git clone https://github.com/Someleze79/My-Portfolio-Website.git
cd My-Portfolio-Website

- Install dependencies:

npm install

- Run the development server:

npm run dev

---

## 🚀 Deployment

### This project is deployed using Vercel.

To deploy:

1. Push your code to GitHub
2. Import the repo into Vercel
3. Deploy instantly
